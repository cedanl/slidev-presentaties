# Images for Clidev Presentation

## Required Images

### 1. Claude Logo
**Filename:** `claude-logo.png` or `claude-logo.svg`
**Source:** https://www.anthropic.com/brand or https://claude.ai
**Size:** 200-300px width recommended
**Usage:** Title slide, "De Echte Kracht" slide

**How to get:**
1. Go to https://claude.ai
2. Right-click on the Claude logo
3. Save image as `claude-logo.png`
4. Place in this folder

### 2. Slidev Logo
**Filename:** `slidev-logo.svg`
**Source:** https://sli.dev
**Size:** 200-300px width
**Usage:** Title slide, "Hoe het werkt" slide

**How to get:**
1. Go to https://sli.dev
2. Right-click on the Slidev logo
3. Save as `slidev-logo.svg`
4. Place in this folder

### 3. Optional: Git/GitHub Screenshots
**Filenames:**
- `git-log-example.png` - Terminal showing git log output
- `gh-issues-example.png` - GitHub issues list
- `generated-presentation.png` - Example of generated slides

**Usage:** "Use Case: Update Presentaties" slide
